What is the Document Object Model (DOM)?

DOM stands for Document Object Model, is a way to do different operations on HTML elements. Such as: updating, fetching or reading, deleting or creatinig new attributes and styles of any HTML elements. It's a general way or method to manipulate HTML's different objects.

How do you select an object from DOM?

There are different kind of selector for different scinarios. Like we can select a unique object with it's id's using getElementById(), or selecting a group of objects by their classes using getElementsByClassName, or selecting various HTML tag or tags using getElementByTagName() etc. Also we can use DOM for CSS manipulation too, using querySelector() to select the first css query, or using querySelectorAll() to select all those similar CSS Queries.

What is an event delegation in the context of DOM, and why is it useful?

Event Delegation is a level-based pattern of handling events in DOM. It works in levels of elements like parent, child, grandchild etc elements.

How do you manipulate and element's attributes and styles using the DOM?

After selecting an element's attributes or styles we can use CSS property names in JS to manipulate them. But as JS does't like "-" sign, we use camel-case to select them, like camelCase. If we want to change background-color of an element, after selecting that element we can use ".styles.backgroundColor: black" to change it to black. Same applies for other CSS styles.
